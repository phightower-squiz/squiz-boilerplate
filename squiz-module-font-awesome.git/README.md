# Font Awesome

[Bower]: http://bower.io/
[Squiz Boilerplate]: https://gitlab.squiz.net/boilerplate/squiz-boilerplate
[Font Awesome]: http://fortawesome.github.io/Font-Awesome

> [Font Awesome][] icon fonts for [Squiz Boilerplate][]

## Overview

[Font Awesome][] is a generic icon font library developed for use, but not restricted to, Twitter Bootstrap. It's a handy library to have around but shouldn't be used in exclusion to custom sprites created by Squiz designers.

## Usage

The best place to see what icons are available is the [official cheatsheet](http://fortawesome.github.io/Font-Awesome/cheatsheet/) page.

```html
<a href="http://squiz.net/"><span class="fa fa-external-link"> An external link</span></a>
```
